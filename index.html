Introduccion a Javascript

¿Qué es un algoritmo?

Un algoritmo es un conjunto de pasos o instrucciones lógicas diseñadas para resolver un problema o realizar una tarea específica.

Datos de entrada / Datos de salida

Los datos de entrada son los valores o informacion que el programa recibe y con los que ejecutara la accion
Los datos de salida son los resultados que el programa arroja despues de procesar los datos de entrada.

¿Qué es JavaScript?

JavaScript es un lenguaje de programación de alto nivel, orientado a objetos. Permite agregar interactividad y dinamismo a las páginas web

JavaScript ES5 y JavaScript Moderno

JavaScript ES5 (ECMAScript 5) es una versión anterior de JavaScript que se utilizaba antes.
JavaScript moderno se refiere a las versiones más recientes de JavaScript, incluyendo ES6 (ECMAScript 2015) y posteriores

¿Qué son los tipos de dato?

Los tipos de datos son categorías que indican el tipo de valor que una variable puede contener.

Tipos de dato en JavaScript

  • Número: Representa valores numéricos, como 5 o 3.14.
    Ejemplo: let age = 15;
  • Cadena de texto: Representa texto entre comillas, pueden ser ' o "
    Ejemplo: let name = 'Ricardo';
  • Booleano: Representa un valor verdadero ('true') o falso ('false').
    Ejemplo: let login = true;
  • Null: Representa un valor vacio.
    Ejemplo: let empty = null;
  • Undefined: Es una variable que ha sido declarada pero aún no ha sido definida.
    Ejemplo: let variableIndefinida;

Diferencias entre 'let', 'var' y 'const'

  • let: Permite declarar variables limitando su alcance al ámbito de bloque .
  • var: Declara una variable global, la cual puede ser llamada en cualquier momento y en cualquier parte. Es más antiguo y menos recomendado que 'let' y 'const'.
  • const: Declara una variable global y su valor no puede ser reasignado una vez que se ha asignado.
